# Проект: Фирменный Стиль и Рекламное Оформление для Рекламного Агентства «ВЕКТОР» 

## Обзор Проекта

Данный проект представляет собой полную разработку фирменного стиля и рекламного оформления для рекламного агентства **«ВЕКТОР»**, ориентированного на малый и средний бизнес (МСБ) с ключевым УТП — **клиентоориентированность** и **прозрачность**.

В отличие от предыдущей итерации, этот проект включает **визуализацию** ключевых элементов стиля.

### Ключевые Элементы Бренда

| Элемент | Описание |
| :--- | :--- |
| **Название** | **«ВЕКТОР»** |
| **Слоган** | «Ваш вектор роста в рекламе и PR» |
| **Специализация** | Реклама и PR для МСБ |
| **Ключевая Ценность** | Клиентоориентированность и Прозрачность |

## Визуальные Активы

### 1. Логотип «ВЕКТОР»

![Логотип «ВЕКТОР»](logo_vektor.png)

*   **Концепция:** Геометрическая стрелка, направленная вверх и вправо.
*   **Цвета:** Глубокий Синий (`#004AAD`) и Яркий Оранжевый (`#FF6600`).

### 2. Мокап Рекламного Оформления

![Мокап Рекламного Баннера](/ad_banner_mockup.png)

*   **Концепция:** Чистый, информативный дизайн с акцентом на Яркий Оранжевый цвет для указания направления и CTA.
*   **Сообщение:** «Ваш вектор роста в рекламе и PR».
