# Креативная Концепция Рекламного Агентства «ВЕКТОР»

## 1. Исходные данные и Фиксация Концепции

| Параметр | Значение |
| :--- | :--- |
| **Название** | **«ВЕКТОР»** |
| **Слоган** | «Ваш вектор роста в рекламе и PR» |
| **Специализация** | Реклама и PR |
| **Целевая аудитория** | Малый и средний бизнес (МСБ) |
| **Ключевое сообщение (УТП)** | Клиентоориентированность и Прозрачность |

## 2. Легенда бренда (Brand Story)

> **«Вектор»** — это агентство, созданное для того, чтобы сделать эффективную рекламу и PR доступными и понятными для малого и среднего бизнеса. Мы верим, что успех наших клиентов — это наш единственный истинный показатель эффективности.
>
> Мы строим свою работу на принципах **абсолютной клиентоориентированности**, что означает глубокое погружение в специфику бизнеса каждого клиента, разработку **индивидуальной стратегии** и **прозрачную отчетность** на каждом этапе. Наша миссия — дать вашему бизнесу **четкий вектор движения** к росту.

## 3. Ключевые ценности

1.  **Клиентоориентированность:** Приоритет интересов и целей клиента.
2.  **Прозрачность:** Открытая коммуникация и честная отчетность.
3.  **Результативность:** Фокус на измеримых показателях роста бизнеса клиента.
4.  **Точность:** Стратегический подход и точное исполнение.
